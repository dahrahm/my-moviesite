My MovieSite is a simple and responsive movie trailer website.
It lets users explore trending movies and instantly watch their trailers in one place.
The goal was to build a clean interface that feels light, fast, and fun to use.

Features
- Browse trending and popular movies
-  Watch official trailers instantly
- Clean and minimal design
- Fully responsive layout for mobile and desktop
- Smooth hover and click effects for better user feel

Tech Stack
- HTML – for structure
- CSS – for styling and layout
- JavaScript – for interactivity and trailer playback
